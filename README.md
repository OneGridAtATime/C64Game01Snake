# C64Game01Snake

Simple Snake-style game built in Unity using the new Input System.

This project is part of a series focused on building small, polished, C64-inspired games with clean structure and repeatable workflows.

---

## 🎯 Purpose

This is **Game 01** in the series.

The goal is not to build a perfect game, but to validate:

* Unity project setup
* Git + GitHub CLI workflow
* Input System (new system only)
* UI Toolkit basics
* Grid-based movement
* Core game loop structure

---

## 🕹 Current Features

* Grid-based movement (Snake-style)
* Direction input (keyboard + controller)
* Movement timer (step-based)
* Basic body following system
* Manual growth trigger (debug)

---

## 🎮 Controls

### Keyboard

* WASD or Arrow Keys → Move

### Controller

* Left Stick or D-Pad → Move

---

## ⚙️ Tech Stack

* Unity 6 (2D)
* New Input System
* UI Toolkit
* Aseprite (placeholder sprites)
